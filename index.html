What is Mode Bridge?

Mode is an OP Stack Layer 2 focused on DeFi and onchain agents; its canonical bridge moves supported assets between Ethereum Mainnet and Mode Mainnet. See the Mode bridge guide for the official deposit and withdrawal framing.

How Mode Bridge works

  1. Open the appUse the official bridge domain listed in Mode network details, then connect a wallet you control.
  2. Select directionChoose a deposit from Ethereum to Mode or a withdrawal from Mode back to Ethereum.
  3. Choose assetPick a supported asset such as ETH, USDC, USDT, or WBTC and enter the amount.
  4. Review and confirmCheck the domain, network, token, amount, gas prompt, and destination before signing.

Before you bridge

Canonical L1 to L2 path

Mode docs describe the canonical bridge as the route for moving funds between Ethereum Mainnet and Mode Mainnet.

Built around supported assets

The user guide lists ETH, USDC, USDT, and WBTC as supported assets, so check the app before sending anything else.

Verify network details

Mode Mainnet uses ETH as its currency symbol and chain ID 34443; confirm against Mode network details before transacting.
